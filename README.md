### The color scheme of the terminal is [Dracula](https://github.com/dracula/dracula-theme)

### Color Scheme:

__Base color__

`white=$(tput setaf 15);`

__Primary colors__

`purple=$(tput setaf 141);`

`pink=$(tput setaf 212);`

`yellow=$(tput setaf 228);`

__Secondary colors__

`cyan=$(tput setaf 51);`

`green=$(tput setaf 84);`

`orange=$(tput setaf 214);`

`red=$(tput setaf 203);`
